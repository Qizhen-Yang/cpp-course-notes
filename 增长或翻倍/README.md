# 增长或翻倍
给定正整数`s`和`t`，计划将`s`一步步地变成`t`。每步操作可以选择：
增长：将`s`变成`s+1`；
或翻倍：将`s`变`2s`。
请求出最少用几步操作可以将`s`变成`t`。

## 输入格式

单独一行：两个整数表示`s`和`t`。

## 输出格式

单独一行：表示最少操作步数。

## 数据范围

1 ≤ `s` ≤ `t` ≤ 1000000000

## 例子
### 1.
Input
```
1 4
```

Output
```
2
```

### 2.
Input
```
2 10
```

Output
```
3
```
