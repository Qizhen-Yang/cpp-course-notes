# 南蛮图腾
有一些少数民族的图腾是分形三角形，你能画出来吗？

## 输入格式

一个整数`n`，表示图腾的大小。

## 输出格式

若干行，表示图腾的样子。
例如：
```
       /\
      /__\
     /\  /\
    /__\/__\
   /\      /\
  /__\    /__\
 /\  /\  /\  /\
/__\/__\/__\/__\
```

## 数据范围

`1 ≤ n ≤ 10`

## 例子
### 1.
Input
```2```

Output
```
   /\
  /__\
 /\  /\
/__\/__\
```

### 2.
Input
```1```

Output
```
 /\
/__\
```

Input
