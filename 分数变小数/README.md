# 分数变小数
给定一个分数P/Q，请将它转化成十进制小数。

- 如果小数部分是有限的，直接输出，如

22/5 = 4.4

- 如果小数部分是0，不输出小数点，如

32/4 = 8

- 如果小数部分是无限循环的，将循环节用圆括号括起来，如

45/56 = 0.803(571428)

## 输入格式
第一行：两个正整数PP和QQ，

## 数据范围
1 ≤ P ≤ 10⁵
1 ≤ Q ≤ 10⁵

### 样例数据 1
Input
```
1 7
```

Output
```
0.(142857)
```

### 样例数据 2
Input
```
8 4
```

Output
```
2
```
